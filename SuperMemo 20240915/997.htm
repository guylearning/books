CSS: Which elements are selected by the space (" ") combinator?

selector1 selector2 {
  /* property declarations */
}

The descendant combinator (a space) selects all elements matching `selector2` that have an ancestor matching `selector1`.

Item #994: Which HTML elements are styled by the CSS code? ul.my-things li { margin: 2em; } (top)

Which HTML elements are styled by the CSS code?

ul.my-things li {
  margin: 2em;
}

All <li> elements (list items) within any <ul> element that has the class `my-things`.
