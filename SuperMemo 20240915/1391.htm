Item #1,488: CSS: What is a font stack?

CSS: What is a font stack?
A list of multiple font names separated by commas, providing fallback options for text display.

body {
  font-family: "Helvetica Neue", Arial, sans-serif;
}

Item #1,485: [True / False]: CSS: Font family names containing spaces, digits, and punctuation characters (other than hyphens) should always be quoted. (top)

[True / False]: CSS: Font family names containing spaces, digits, and punctuation characters (other than hyphens) should always be quoted.
True.

body {
  font-family: "Helvetica Neue", Arial, sans-serif;
}

Item #1,486: [True / False]: CSS: Generic font family names should be quoted when used as keywords. (top)

[True / False]: CSS: Generic font family names should be quoted when used as keywords.
False. MUST NOT be quoted.

body {
  font-family: "Helvetica Neue", Arial, sans-serif;
}

Item #1,487: CSS: How to prevent the browser from using an inappropriate default font (such as a serif font) for a sans-serif design? (top)

CSS: How to prevent the browser from using an inappropriate default font (such as a serif font) for a sans-serif design?
To specify the `font-family` property with a suitable generic font name at the end of the font stack.

body {
  font-family: "Helvetica Neue", Arial, sans-serif;
}
