CSS: What is the function of the `i` flag in an attribute selector?

li[class^="a" i] {
  color: red;
}

To force case-insensitive matching.

Item #1,209: CSS: Where is the `i` flag placed in an attribute selector? (top)

CSS: Where is the `i` flag placed in an attribute selector?
Before the `]` closing bracket.

For example:

li[class^="a" i] {
  color: red;
}
