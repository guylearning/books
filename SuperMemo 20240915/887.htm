CSS type selectors are [case-sensitive / case-insensitive].

Will `p` and `P` in the CSS selectors target to different elements?

  • p {
      color: blue;
    }
  • P {
      color: red;
    }
Case-insensitive.

The selectors p and P will target the same elements.
