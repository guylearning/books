Item #801: What is the syntax for a CSS declaration? (top)

What is the syntax for a CSS declaration?
property: value;

A CSS declaration consists of a property and a value, separated by a colon and a space, and ending with a semicolon.

Item #805: CSS: Properties are [case-sensitive / case-insensitive] by default. (top)

CSS: Properties are [case-sensitive / case-insensitive] by default.
Case-insensitive.

Properties like `background-color`, `Background-Color`, and `BACKGROUND-COLOR` are all interpreted the same way by the browser.

Item #806: CSS: Values are [case-sensitive / case-insensitive] by default. (top)

CSS: Values are [case-sensitive / case-insensitive] by default.
Case-insensitive.

CSS keywords (e.g., color, background-color) are case-insensitive. However, certain values like URLs and font names are case-sensitive. #do_research_more
