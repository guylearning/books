Topic #1,670: Never point the src attribute at an image hosted on someone else's website without permission. This is called "hotlinking". It is considered unethical, since someone else . (top)

Never point the src attribute at an image hosted on someone else's website without permission. This is called "hotlinking". It is considered unethical, since someone else would be paying the bandwidth costs for delivering the image when someone visits your page. It also leaves you with no control over the image being removed or replaced with something embarrassing.

What is hotlinking in internet terms?
Using a direct web link to display an asset (like an image, video, or audio file) hosted on another website without permission.

A hotlink (also known as an inline link) is an object (typically an image) directly linked to from another site. For example, an image hosted on site1.com is shown directly on site2.com.

What is the term for displaying an asset using a direct link from another website without permission?
Hotlinking.

When an image is hotlinked, who pays for the bandwidth costs, the image owner or the hotlinker?
The image owner.

Hotlinking is considered unethical, since someone else would be paying the bandwidth costs for delivering the image when someone visits your page.

Item #1,633: What control-related issues can arise from using hotlinked images? (top)

What control-related issues can arise from using hotlinked images?
The image could be removed or replaced with something embarrassing.

Item #1,634: Is hotlinking considered ethical or unethical? (top)

Is hotlinking considered ethical or unethical?
Unethical.

The practice is frowned upon, as it can cause unwanted bandwidth usage on the website hosting the linked-to object. From an ethical standpoint, it could be considered stealing when done without permission.

Topic #1,671: Elements like and (top)

Elements like <img> and <video> are sometimes referred to as replaced elements. This is because the element's content and size are defined by an external resource (like an image or video file), not by the contents of the element itself. You can read more about them at Replaced elements.

Topic #1,672: Alternative text (top)

Is the alt attribute in an <img> HTML element read by screen readers? Yes.
The alt attribute provides descriptive text for images, which screen readers read aloud to assist visually impaired users.
