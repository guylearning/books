    Item #1,156: Which HTML elements are styled by the CSS code? li[special] { font-size: 200%; } (top)

    Which HTML elements are styled by the CSS code?

    li[special] {
      font-size: 200%;
    }
    All <li> (list item) elements that have the `special` attribute.

    In the given HTML code, “Item 2” and “Item 3” will have a font size of 200%, while “Item 1” will remain unaffected.

    <ul>
      <li>Item 1</li>
      <li special="a">Item 2</li>
      <li special="a b">Item 3</li>
    </ul>
