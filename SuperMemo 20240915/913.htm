    Item #910: CSS class selectors are [case-sensitive / case-insensitive]. Will `class1` and `Class1` in the CSS selectors target to different elements? class1 { color: blue; } Class1 . (top)

    CSS class selectors are [case-sensitive / case-insensitive].

    Will `class1` and `Class1` in the CSS selectors target to different elements?

    • class1 {
        color: blue;
      }
    • Class1 {
        color: red;
      }
    Case-sensitive.

    The selectors class1 and Class1 will target different elements.

    Which HTML elements are styled by the CSS code?

    .notebox.danger {
      border-color: red;

    }

    All elements with a class list that includes both the class `notebox` and the class `danger`.

    For example:

    <div class="notebox danger">
      This note shows danger!
    </div>

    Item #914: CSS: Do the selectors `.class1.class2` and `.class2.class1` target different elements? (top)

    CSS: Do the selectors `.class1.class2` and `.class2.class1` target different elements?
    No. In CSS class selectors, the order of chained class selectors doesn't matter.
