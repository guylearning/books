Item #1,511: Which typed language is JavaScript?

Which typed language is JavaScript?
A dynamically typed language.

Item #1,512: [True / False]: JavaScript: A variable can be reassigned from an integer to a string without explicit type declarations. (top)

[True / False]: JavaScript: A variable can be reassigned from an integer to a string without explicit type declarations.
True.

JavaScript is a dynamically typed language, which means variables can hold different types of data throughout their lifetime without requiring explicit type declarations. When a variable is reassigned, JavaScript automatically handles the type change. For example, let x = 5; (integer) can be followed by x = "hello"; (string) without any additional type conversion or declaration.
