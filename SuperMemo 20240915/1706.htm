Item #1,700: HTML: In an element, what is the function of the `title` attribute?

HTML: In an <img> element, what is the function of the `title` attribute?
To provide additional information about the image as a tooltip.

Item #1,703: [True / False]: HTML: It is best practice to use the `title` attribute for extra image information. (top)

[True / False]: HTML: It is best practice to use the `title` attribute for extra image information.
False.

The title attribute has a number of accessibility problems. Instead, use the alt attribute for essential information and accessibility, and consider using captions or adjacent text for additional details.
