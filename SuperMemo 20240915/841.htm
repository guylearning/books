Item #809: What happens to a CSS declaration if the property is unknown?

What happens to a CSS declaration if the property is unknown?
The declaration is ignored by the browser's CSS engine.

The browser skips the invalid declaration and continues rendering the rest of the CSS rules.

CSS fallback styling.

CSS fallback refers to providing alternative styles for browsers that do not support certain CSS features. This ensures that a webpage remains functional and visually consistent across different browsers and versions. For example:

.box {
  width: 500px; /* Fallback width for older browsers */
  width: calc(100% - 50px); /* Modern browsers */
}
