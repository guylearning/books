Item #1,023: CSS: Which elements are selected by the ~ combinator? selector1 ~ selector2 { /* property declarations */ }

CSS: Which elements are selected by the ~ combinator?

selector1 ~ selector2 {
  /* property declarations */
}

The subsequent-sibling combinator (~) selects all elements matching `selector2` that follows the element matching `selector1` (not necessarily immediately) and share the same parent element.

Item #1,024: Which HTML elements are styled by the CSS code? img ~ p { color: red; } (top)

Which HTML elements are styled by the CSS code?

img ~ p {
  color: red;
}

Paragraphs that are both siblings of an image and appear after any image.

CSS: The white space around the ~ combinator is [required/optional].

former_element ~ target_element {
  /* property declarations */
}

Optional, but recommended.
