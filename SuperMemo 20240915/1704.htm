Item #1,698: HTML: Is the `alt` attribute in an element read by screen readers? (top)

HTML: Is the `alt` attribute in an <img> element read by screen readers?
Yes.

The alt attribute provides descriptive text for images, which screen readers read aloud to assist visually impaired users.
