Item #1,141: What happens to a CSS ruleset with an invalid or unsupported selector in the selector list?. h1, .special {. color: blue;. } (top)

What happens to a CSS ruleset with an invalid or unsupported selector in the selector list?

h1, ..special {
  color: blue;
}

The entire ruleset is ignored.

Item #1,142: [True / False]: CSS: The browser skips an invalid selector in a selector list and continues rendering the rest of the valid selectors. (top)

[True / False]: CSS: The browser skips an invalid selector in a selector list and continues rendering the rest of the valid selectors.
False.

If any part of a selector list is invalid, the entire rule is ignored by the browser.

Which HTML elements are styled by the CSS code?

h1,
..special {
  color: blue;
}

No HTML elements.

The provided CSS code contains a syntax error due to the invalid selector `..special`. Because of this, the entire rule is ignored.
