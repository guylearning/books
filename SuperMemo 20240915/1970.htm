    HTML: What is the function of the <caption> element?
    To describe the contents of the table.

    Item #1,944: HTML: Where is the

    element placed in an element? (top)

    HTML: Where is the
    element placed in an element?
    As the first child of the <table> element.

    The <caption> element is placed Immediately after the opening <table> tag and before any other table content (such as <thead>, <tbody>, or <tr>).
