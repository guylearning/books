[True / False]: HTML: It is possible to nest multiple levels of lists.
True. Nesting lists is allowed.

      HTML: In nesting lists, which element is a nested <ul> or <ol> list placed?
      Inside a <li> (list item) element of the parent list.

        Item #1,612: Is the following HTML code correctly nesting an ordered list within an unordered list?

        • Item 1
        • Item 2
          1. Subitem 1
            2. .             (top            )

            HTML: Is the given code correctly nesting an ordered list within an unordered list?

            <ul>
              <li>Item 1</li>
              <li>Item 2</li>
                <ol>
                  <li>Subitem 1</li>
                  <li>Subitem 2</li>
                </ol>
              <li>Item 3</li>
            </ul>

            Incorrect.

            The ordered list (<ol>) should be nested inside a list item (<li>) of the unordered list (<ul>). Corrected HTML code:

            <ul>
              <li>Item 1</li>
              <li>Item 2
                <ol>
                  <li>Subitem 1</li>
                  <li>Subitem 2</li>
                </ol>
              </li>
              <li>Item 3</li>
            </ul>
