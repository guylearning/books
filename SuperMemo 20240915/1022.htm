Item #1,014: CSS: Which elements are selected by the + combinator? selector1 + selector2 { /* property declarations */ }

CSS: Which elements are selected by the + combinator?

selector1 + selector2 {
  /* property declarations */
}

The next-sibling combinator (+) selects the first element matching `selector2` that directly follows the element matching `selector1` and share the same parent element.

Item #1,015: Which HTML elements are styled by the CSS code? img + p { font-weight: bold; } (top)

Which HTML elements are styled by the CSS code?

img + p {
  font-weight: bold;
}

Paragraphs (<p> elements) that are both siblings and immediately follow any image (<img> element).
