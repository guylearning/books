Which HTML component provides additional information about an element?
Attributes.

What is the function of HTML attributes?
To provide additional information about elements.

Item #506: What is the syntax for writing an HTML attribute? (top)

What is the syntax for writing an HTML attribute?
attribute_name="attribute_value"

A name-value pair, separated by "=".

Item #507: In which part of an HTML element are attributes located? (top)

In which part of an HTML element are attributes located?
The opening tag.

Item #508: Which symbol is used to separate multiple attributes in an HTML element? (top)

Which symbol is used to separate multiple attributes in an HTML element?
Space.

For example: <a href="https://example.com" class="link" id="example-link">Example</a>

Item #509: Which symbol separates an attribute from the element name in an HTML opening tag? (top)

Which symbol separates an attribute from the element name in an HTML opening tag?
Space.

For example: <p class="editor-note">My cat is very grumpy</p>

Item #510: What is wrong with the given HTML syntax: My cat is very grumpy

(top)

What is wrong with the given HTML syntax:

<class="editor-note" p>My cat is very grumpy</p>

<p class="editor-note">My cat is very grumpy</p>

HTML attributes cannot be placed between the < symbol and the element name in the opening tag. HTML attributes must be placed within the opening tag, after the element name.

Item #512: [True / False]: Commas (,) can be used as separators for multiple attributes in HTML elements. . (top)

[True / False]: Commas (,) can be used as separators for multiple attributes in HTML elements.

<a
  href="https://example.com",
  title="Example Site",
  target="_blank">
  Visit Example
</a>

False. Use spaces.

<a href="https://example.com" title="Example Site" target="_blank">Visit Example</a>

In HTML, commas (,) cannot be used as separators for multiple attributes in HTML elements. Attributes should be separated by spaces.

Item #521: When is it permissible to omit quotes around attribute values in HTML? (top)

When is it permissible to omit quotes around attribute values in HTML?
Attribute values must not contain spaces, tabs, or special characters like =, <, >, or ".

Item #522: [True / False]: Changing the order of attributes in an HTML element will alter its behavior. (top)

[True / False]: Changing the order of attributes in an HTML element will alter its behavior.
False. The attribute order in HTML elements is not important.

Item #519: Which quotes are correct for HTML attribute values, [single (') / double (")]?. A link to my example.. top)

Which quotes are correct for HTML attribute values, [single (') / double (")]?
(Trick question) Both single quotes (') and double quotes (") are correct. However, double quotes (") are more commonly recommended.

Both of the lines are equivalent:

  • <a href='https://www.example.com'>A link to my example.</a>
  • <a href="https://www.example.com">A link to my example.</a>

Item #520: [True / False]: Mixing single and double quotes for enclosing an HTML attribute's value is valid syntax. top)

[True / False]: Mixing single and double quotes for enclosing an HTML attribute's value is valid syntax.

  • <a href="https://www.example.com'>A link to my example.</a>
False.

Use either single quotes or double quotes consistently for each attribute value.

  • <a href="https://www.example.com">A link to my example.</a>
  • <a href='https://www.example.com'>A link to my example.</a>
